# 总体设计

## 管理目标
- 预算前置
- 费用归项目
- 审核有依据
- 风险可识别
- 结果可核销
- 数据可沉淀

## 适用范围
- 活动项目
- 市场项目
- 人事专项项目
- 行政采购项目
- 跨部门协同项目

## 核心管理对象
- 项目
- 预算
- 费用
- 发票
- 报销单
- 审核节点
- 核销状态

## 总体流转主线
项目立项
-> 预算编制
-> 预算审核
-> 预算启用
-> 费用发生
-> 发票收集
-> 报销草稿
-> 项目关联
-> 费用分摊
-> 业务审核
-> 项目审核
-> 财务审核
-> 阶段性控制
-> 报销完成
-> 核销
-> 归档
-> 复盘
