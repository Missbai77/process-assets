# 总体设计

## 管理目标
- 建立非项目制经营支出管理框架
- 区分成本类支出与期间费用类支出
- 统一预算、申请、审核、入账、核销和结转逻辑
- 提高支出归类准确性和财务可追踪性
- 为经营分析和费用复盘提供基础数据

## 适用范围
- 采购类支出
- 生产制造类支出
- 委外加工类支出
- 销售费用类支出
- 管理费用类支出
- 研发费用类支出
- 财务费用类支出
- 其他非项目制经营支出

## 核心管理对象
- 成本类支出
- 期间费用类支出
- 直接材料
- 直接人工
- 制造费用
- 销售费用
- 管理费用
- 研发费用
- 财务费用
- 发票
- 业务单据
- 审核节点
- 入账状态
- 核销状态

## 管理口径说明
- 会计核算按财务科目执行
- 支出先区分成本类与期间费用类
- 成本类支出重点关注归集、分配、入库和结转
- 期间费用重点关注合规性、必要性和归类准确性
- 管理分析可进一步看与销售收入的关系，但不替代会计归类

## 总体流转主线
预算建立
-> 支出发生
-> 单据与发票收集
-> 成本 / 费用归类
-> 业务审核
-> 财务审核
-> 入账
-> 核销 / 对账
-> 结转 / 归档
-> 复盘
