# 通用经营支出管理

这是一个适用于**非项目制**场景的经营支出管理案例。

重点不是只看“报销流程”，而是围绕企业日常经营中的支出活动，建立一套从预算、支出发生、归类审核、入账、核销到结转复盘的基础管理框架。

---

## 一、案例目标

- 建立非项目制经营支出管理框架
- 区分成本类支出与期间费用类支出
- 统一预算、申请、审核、入账、核销和结转逻辑
- 提高支出归类准确性和财务可追踪性
- 为经营分析和费用复盘提供基础数据

---

## 二、适用范围

适用于非项目制企业日常经营支出，例如：

- 采购类支出
- 生产制造类支出
- 委外加工类支出
- 销售费用类支出
- 管理费用类支出
- 研发费用类支出
- 财务费用类支出
- 其他日常经营支出

---

## 三、管理口径

### 1. 会计核算口径
支出先区分为两类：

- **成本类**
  - 直接材料
  - 直接人工
  - 制造费用

- **期间费用类**
  - 销售费用
  - 管理费用
  - 研发费用
  - 财务费用

### 2. 管理分析口径
在不替代会计归类的前提下，可进一步分析支出与销售收入的关系。
