# Process Assets

用于沉淀流程数字化、低代码系统设计和业务标准化经验的个人资产库。

## 仓库结构

- `expense/`：费用专题
- `GOAL.md`：当前阶段目标记录

## expense 专题内容

- `cases/`：费用管理案例
- `methods/`：费用管理方法
- `templates/`：费用管理模板

## 已完成案例

### `expense/cases/biz-expense/`
非项目制通用经营支出管理。

### `expense/cases/biz-expense-p/`
项目型费用管理体系。

## 仓库目标

- 展示流程数字化与业务分析能力
- 沉淀可复用的案例、方法和模板
- 逐步扩展更多业务专题
- 为作品集和独立输出打基础

## 阅读顺序

1. `GOAL.md`
2. `expense/README.md`
3. `expense/cases/`
4. `expense/methods/`
5. `expense/templates/`
