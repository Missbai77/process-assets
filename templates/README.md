# Templates

这个文件夹用来保存流程数字化相关模板。

模板的作用是让经验可以被重复使用，而不是每次从空白开始。

## 当前模板

- `process-case-collection-template.md`：流程需求收集模板
- `process-check-template.md`：流程检查模板
- `process-case-design-template.md`：流程设计模板
- `process-case-template.md`：流程案例沉淀模板，可用于基础通用案例和业务案例

## 使用顺序

1. 用需求收集模板记录业务方的问题和期待
2. 用流程检查模板判断当前卡点和风险
3. 用流程设计模板把流程转成字段、节点、规则和系统承接方式
4. 用案例模板沉淀最终经验
