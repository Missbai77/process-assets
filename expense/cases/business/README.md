# Business Cases

费用专题下的业务案例。

用于沉淀完整的费用管理场景案例，与基础问题案例相对应。

## 当前案例

### `expense-general/`
非项目制通用经营支出管理。

适用于不以项目为归集单位的经营支出场景，重点围绕预算、支出发生、归类审核、入账、核销、结转和复盘展开。

### `expense-project/`
项目型费用管理体系。

适用于以项目为归集单位的费用管理场景，重点围绕项目立项、预算启用、费用发生、项目关联、分摊、审核、核销和复盘展开。

## 作用

- 展示两类费用管理场景的完整案例
- 区分非项目制与项目型费用管理逻辑
- 为后续方法和模板沉淀提供案例基础

## 阅读顺序

1. `expense-general/`
2. `expense-project/`
