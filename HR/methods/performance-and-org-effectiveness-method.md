# 绩效与组织效能管理方法

## 方法目标

- 让绩效评价有统一口径
- 让个人、部门、项目评价分层管理
- 让绩效结果与组织调整联动
- 让组织效能能够数据化分析

## 方法主线

```text
目标确认
-> 指标制定
-> 过程跟踪
-> 结果评估
-> 绩效复盘
-> 组织效能分析
-> 招聘与编制联动
-> 组织优化调整
```

## 核心管理逻辑

### 个人绩效是否真实有效

需要确认：

- 是否完成目标
- 是否具备结果输出
- 是否存在无效忙碌
- 是否具备协同能力
- 是否具备复用价值

### 部门绩效是否合理

需要确认：

- 部门整体效率
- 人员投入产出
- 协同效率
- 流程效率
- 成本控制情况

### 项目效能是否成立

需要确认：

- 项目目标
- 项目周期
- 人员投入
- 成本投入
- 最终产出
- ROI情况

### 招聘与编制是否合理

需要确认：

- 是否真实缺人
- 是否存在低效流程
- 是否属于阶段性波峰
- 是否可通过优化解决
- 是否需要新增编制

## 方法特点

- 区分个人、部门、项目三层评价
- 绩效与组织效能联动
- 绩效与招聘需求联动
- 强调结果与效率
- 支持组织优化分析

## 适用场景

- 常规绩效管理
- 组织效能分析
- 编制控制场景
- 人效优化场景
- AI协同组织场景
