# 薪酬与人员成本管理方法

## 方法目标

- 让薪酬核算有统一口径
- 让人员成本归属清晰
- 让分摊逻辑可追溯
- 让核算结果与组织数据联动

## 方法主线

```text
核算口径确认
-> 基础数据收集
-> 薪酬核算
-> 成本归集
-> 成本分摊
-> 对账复核
-> 结果归档
-> 数据分析
```

## 核心管理逻辑

### 薪酬数据是否统一

需要确认：

- 数据来源是否统一
- 考勤数据是否一致
- 绩效数据是否同步
- 社保公积金是否正确
- 个税数据是否匹配

### 人员成本是否可归属

需要确认：

- 部门归属
- 项目归属
- 成本中心
- 门店归属
- 多项目人员分摊逻辑

### 分摊逻辑是否明确

需要确认：

- 分摊比例
- 分摊周期
- 工时依据
- 项目依据
- 特殊场景处理方式

### 对账结果是否一致

需要确认：

- 应发与实发
- 财务数据
- 银行数据
- 个税数据
- 历史版本记录

## 方法特点

- 薪酬与成本统一管理
- 数据来源统一
- 分摊逻辑标准化
- 核算全过程留痕
- 支持后续经营分析

## 适用场景

- 常规薪酬核算
- 项目成本管理
- 多部门成本分摊
- 财务共享场景
- 流程数字化建设场景
