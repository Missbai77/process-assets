# HR

人事管理专题。

用于沉淀招聘、入职、试用期、转正、调岗、调薪、离职、薪酬核算、人员成本分摊和人员档案管理相关的案例、方法和模板。

## 目录结构

- `cases/`：人事管理案例
- `methods/`：人事管理方法
- `templates/`：人事管理模板

## 管理主线

人事管理不是单一流程，而是一条贯穿员工全周期的管理链路：

```text
招聘需求提出
-> 招聘需求有效性判断
-> 招聘执行
-> 筛选简历
-> 面试评估
-> 录用判断
   -> 不通过：进入人才库
   -> 通过：进入入职流程
-> 入职
-> 试用期 / 转正
-> 调岗 / 调薪
-> 离职
