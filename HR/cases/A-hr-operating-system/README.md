# 人事运营体系

这是一个围绕招聘需求提出、招聘需求有效性判断、招聘执行、面试评估、人才库、入职衔接、员工生命周期、绩效、薪酬核算、人员成本分摊和人员档案管理展开的综合案例。

## 一、案例目标

- 招聘需求有依据
- 招聘执行有节奏
- 面试评估有记录
- 入职衔接无断点
- 员工状态全周期可跟踪
- 绩效、薪酬、成本和档案可联动

## 二、适用范围

- 常规组织人事管理场景
- 有招聘、入职、异动、离职管理需求的团队
- 需要人效、薪酬和成本联动管理的组织

## 三、整体结构

```text
事前：需求提出与有效性判断
-> 事中：招聘执行、面试、录用与入职衔接
-> 事后：在职管理、绩效、薪酬成本与闭环
