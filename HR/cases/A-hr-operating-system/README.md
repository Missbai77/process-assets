# 人事运营体系

这是一个围绕招聘需求提出、招聘需求有效性判断、招聘执行、面试评估、人才库、入职衔接、员工生命周期、绩效、薪酬核算、人员成本分摊和人员档案管理展开的综合案例。

## 一、案例目标

- 招聘需求有依据
- 招聘执行有节奏
- 面试评估有记录
- 入职衔接无断点
- 员工状态全周期可跟踪
- 绩效、薪酬、成本和档案可联动

## 二、适用范围

- 常规组织人事管理场景
- 有招聘、入职、异动、离职管理需求的团队
- 需要人效、薪酬和成本联动管理的组织
- 需要把招聘、人效、薪酬成本和档案统一管理的组织

## 三、整体结构

```text
事前：招聘需求有效性判断
-> 事中：招聘执行、面试、录用与入职衔接
-> 事后：员工生命周期、薪酬成本与档案联动
-> 风险识别与控制
```

## 五、管理主线

```text
用人需求提出
-> 需求信息确认
-> 需求有效性判断
-> 招聘执行
-> 简历筛选
-> 面试评估
-> 结果分流
-> Offer确认
-> 入职衔接
-> 员工生命周期管理
-> 薪酬成本联动
-> 档案归档
-> 风险复盘
```

## 六、核心价值

- 把招聘从执行动作前置为组织判断
- 把候选人管理沉淀为人才资产
- 把入职、转正、调岗、调薪、离职纳入员工生命周期
- 把员工状态与薪酬、成本、权限、档案联动
- 把人事流程从线性审批转为可追踪、可复盘的管理系统
