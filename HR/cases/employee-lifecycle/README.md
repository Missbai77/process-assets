# 员工生命周期管理

用于沉淀员工从入职、试用期、转正、调岗、调薪到离职的完整管理链路。

## 文件结构

- `01-overview.md`：总体设计
- `02-entry.md`：入职管理
- `03-probation-and-regularization.md`：试用期与转正
- `04-job-change-salary-change-exit.md`：调岗、调薪与离职
- `05-risk-and-control.md`：风险识别与控制

## 管理主线

```text
录用确认
-> 入职准备
-> 到岗入职
-> 试用期跟踪
-> 转正判断
-> 在职异动
-> 调岗 / 调薪
-> 离职交接
-> 档案收口
