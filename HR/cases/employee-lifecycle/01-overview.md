# 总体设计

## 管理目标与原则

- 员工入职前准备完整
- 入职过程不断点
- 试用期有跟踪
- 转正有判断依据
- 调岗、调薪、离职有流程记录
- 员工档案随状态变化同步更新
- 员工状态变化必须留痕
- 入职、转正、异动、离职不能割裂
- 档案不是最后补，而是全流程同步更新
- 薪酬、权限、组织关系要和员工状态同步

## 核心对象

- 员工基础信息
- 入职资料
- 劳动合同
- 试用期记录
- 转正结果
- 调岗记录
- 调薪记录
- 离职记录
- 员工档案

## 流程主线

```text
录用确认
-> 入职准备
-> 入职办理
-> 试用期管理
-> 转正管理
-> 在职异动
-> 离职管理
-> 档案归档
