# Cases

人事专题下的案例目录。

用于沉淀完整的人事管理场景案例。

## 当前案例方向

### `A-hr-operating-system/`

人事运营体系总案例。

围绕招聘需求提出、招聘需求有效性判断、招聘执行、面试评估、人才库、入职衔接、员工生命周期、薪酬成本和档案管理展开。

### `B-employee-lifecycle/`

员工生命周期管理案例。

围绕入职、试用期、转正、调岗、调薪、离职和档案更新展开。

### `C-payroll-and-cost/`

薪酬核算与人员成本管理案例。

围绕薪酬核算、人员成本归集、成本分摊、对账与归档展开。

### `D-performance-and-org-effectiveness/`

绩效与组织效能管理案例。

围绕个人绩效、部门绩效、项目绩效、招聘需求有效性和编制逻辑展开。

## 阅读顺序

1. `A-hr-operating-system/`
2. `B-employee-lifecycle/`
3. `C-payroll-and-cost/`
4. `D-performance-and-org-effectiveness/`
