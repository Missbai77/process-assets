# Cases

人事专题下的案例目录。

用于沉淀完整的人事管理场景案例。

## 当前案例

### `hr-operating-system/`
人事运营体系主案例。

围绕招聘需求、招聘执行、面试评估、入职衔接、人才库、人员档案、绩效、异动和离职展开。

### `employee-lifecycle/`
员工生命周期管理案例。

围绕入职、试用期、转正、调岗、调薪、离职和档案更新展开。

### `payroll-and-cost/`
薪酬核算与人员成本管理案例。

围绕薪酬核算、人员成本归集、成本分摊、对账与归档展开。
